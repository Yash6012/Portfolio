# Yash Singh | Professional Portfolio

## Overview
A modern, responsive portfolio website showcasing my work as a Computer Science Engineering student specializing in IoT Systems, Raspberry Pi Solutions, and Software Engineering. Designed to highlight technical expertise with a polished, engineering-focused aesthetic.

#### Link - https://yash6012.github.io/Portfolio/

## Key Features
- **Responsive Design**: Fluid layout that adapts seamlessly to desktop, tablet, and mobile screens.
- **Dynamic Themes**: Toggle between a clean Light Mode and an "Engineering Dark Mode", with preferences saved automatically.
- **Interactive UI**:
  - **Typing Effect**: Dynamic subtitle animation in the Hero section.
  - **Scroll Animations**: Sections and elements fade in and lift as you scroll.
  - **ScrollSpy**: Navigation menu automatically highlights the active section.
- **Performance Optimized**: Merged CSS and JS files for reduced HTTP requests and faster load times.
- **Project Showcase**: Detailed project cards with custom imagery and tech stack tags.

## Directory Structure
```
portfolio/

├── css/                # styles.css (Merged styles)
├── js/                 # main.js (Merged logic)
├── index.html          # Entry point
└── README.md           # Project documentation
```

## Getting Started
1. **Clone or Download** the repository.
2. **Open** `index.html` in any modern web browser to view the site.
3. **Local Development**: For the best experience, run using a local development server (e.g., Live Server extension in VS Code) to ensure all assets load correctly.
